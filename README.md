# カオゼロ 神ヒラメキ記録ツール

カオスゼロナイトメア（日本版）の神ヒラメキ情報を効率よく記録するためのツール群です。  
**3つのツールを組み合わせて使います。**

-----

## ゲーム用語の説明

|用語       |説明                                             |
|---------|-----------------------------------------------|
|ヒラメキ     |戦闘中にカードが変化する強化システム。通常ヒラメキ（1〜5）と隠れヒラメキ（6）がある    |
|神ヒラメキ    |ヒラメキ発生時に5神のいずれかのアイコンが出現し、追加効果が乗る上位システム         |
|5神       |ディアロス・セクレド・キルケン・ニヒルム・ヴィトルの5種類。それぞれ固有のアイコンカラーを持つ|
|ヒラメキ番号   |各カードのヒラメキを識別する記号（例：A-3、C-5）。A〜DはカードID、数字はパターン番号|
|隠れヒラメキ   |パターン6（X-6）。基本カード効果を保ったまま追加効果のみが乗る特殊パターン        |
|神ヒラメキ追加効果|神様アイコンの後に表示される効果テキスト。元のヒラメキ効果とは別に記録する          |

-----

## システム構成

```
┌─────────────────────────────────────────────┐
│  STEP 1           STEP 2           STEP 3   │
│  iOSショートカット → 神様識別アプリ → Claude │
│  （テキスト抽出）  （アイコン識別） （記録）  │
└─────────────────────────────────────────────┘
```

### 各ツールの役割

|ツール       |役割             |実行場所                |
|----------|---------------|--------------------|
|iOSショートカット|カード効果テキストの抽出   |iPhoneのショートカットアプリ   |
|神様識別アプリ   |神様アイコンの自動識別    |Safari（GitHub Pages）|
|Claude    |ヒラメキ番号の判定・PDF記録|Claude.ai           |

-----

## 必要なもの

- iPhone（iOSショートカットの実行に必要）
- Safariブラウザ（神様識別アプリの使用に必要）
- Claude.aiのアカウント（記録作業に必要）

-----

## 撮影する画像について

使用する画像は以下のいずれかでOKです。

- **詳細画面のフルスクリーンショット（推奨）**：神ヒラメキ発動中に詳細ボタンを押した画面
- **カード拡大画像**：カード部分のみが写っていれば代用可

端末の画面サイズが異なっても、ツールは比率ベースで処理するため問題ありません。

-----

## ツール1：iOSショートカット

iPhoneのショートカットアプリで動作します。  
**自作が必要です。** 以下のアクションを順番に設定してください。

### アクション一覧

|# |アクション             |設定内容                                                     |
|--|------------------|---------------------------------------------------------|
|1 |写真を選択             |複数選択を許可する                                                |
|2 |写真の各項目を繰り返す       |—                                                        |
|3 |繰り返し項目を切り取り       |カード名＋カード分類エリアに合わせて範囲指定                                   |
|4 |切り取り済みの画像からテキストを抽出|—                                                        |
|5 |繰り返し項目を切り取り       |カード効果テキストエリアに合わせて範囲指定                                    |
|6 |切り取り済みの画像からテキストを抽出|—                                                        |
|7 |テキスト              |内容：`繰り返しインデックス.` + 改行 + `===` + 改行 + `4の結果` + 改行 + `6の結果`|
|8 |テキストを変数に追加        |変数名：神ヒラメキテキスト出力結果                                        |
|9 |繰り返しの終了           |—                                                        |
|10|変数を改行で結合          |神ヒラメキテキスト出力結果を改行で結合                                      |
|11|結合済みのテキストをアプリに保存  |保存先アプリを指定                                                |

### 画像の切り取り範囲について

アクション3・5の切り取り範囲は端末サイズに合わせて手動で調整してください。  
切り取る箇所は以下の2箇所です——

- **アクション3**：カード名とカード分類（攻撃/スキル/強化）が写る上部エリア
- **アクション5**：カード効果テキスト全体が写る下部エリア（神ヒラメキ追加効果も含む）

### 出力形式

```
#1.
##==カード名・種別==
ブラックホール
攻撃
##==カード効果・ヒラメキ==
ランダムな敵にダメージ72%
消滅した虚無数に応じてヒット数1回追加（最大5回）
対象に苦痛4

===next===

#2.
##==カード名・種別==
虚妄の誓約
強化
##==カード効果・ヒラメキ==
...

===next===
```

### 注意点

- ショートカットはiPhoneのホーム画面に追加して使用します
- 画像の選択順がそのまま番号になります
- **神様識別アプリと同じ順番で画像を選択してください**
- OCRの精度上、テキストに誤認識が含まれることがあります
- 神ヒラメキアイコン部分はノイズ文字に化けやすい（記録時に要確認）

## ツール2：神様識別アプリ

Safari で以下のURLにアクセスして使用します。  
（`your-username` の部分をご自身のGitHubユーザー名に置き換えてください）

```
https://your-username.github.io/-/
```

### 使い方

1. **STEP 1**：カメラロールから画像を選択（複数可・20枚以下推奨）
- ショートカットと**同じ順番**で選択すること
1. **STEP 2**：識別結果を確認・修正
- プルダウンで神様を変更できます
- スコア差が小さい場合は⚠マークが出ます（目視確認推奨）
- プレビュー画像をタップで拡大確認できます
1. **STEP 3**：ショートカットで出力したテキストファイルをアップロード
- **⚡ 合体して出力** で番号照合・結合
- 結果をコピーまたはテキストファイルとして保存

### 自動識別の仕組み

各画像から神様アイコン領域を切り出し、5神の代表色との色距離を計算して識別します。  
スコア差3.0未満の場合は信頼度が低いため目視確認を推奨します。

## ツール3：Claude（または手動）

以下の作業は**人間が手動で行うことも可能**です。  
Claudeを使うことで作業を効率化できます。

### Claudeへの入力形式

アプリの合体結果をそのままClaudeに貼り付けます。

```
#1.
##==神の名前==
【ニヒルム】
##==カード名・種別==
ブラックホール
攻撃
##==カード効果・ヒラメキ==
ランダムな敵にダメージ72%
消滅した虚無数に応じてヒット数1回追加（最大5回）
→ 対象に苦痛4
===next===
#2.
##==神の名前==
【ヴィトル】
...
```

### 担当する作業

|作業         |内容                                   |
|-----------|-------------------------------------|
|ヒラメキ番号の判定  |抽出テキストを手元のリストと見比べてA-3・C-5などのナンバーを特定する|
|隠れヒラメキの判定  |X-6かどうかの判断（基本効果＋追加効果の構造かどうかを確認）      |
|キャラクター別振り分け|カード名からどのキャラのどのカードかを特定する              |
|PDFへの記録    |神ヒラメキリストのPDFに情報を追加する                 |

### PDF記録について

PDF化・整形はClaudeが得意な作業です。  
フォーマット維持の観点からClaudeの使用を推奨します。

-----

## 全体ワークフロー

```
① スクリーンショットを撮る
   （神ヒラメキ発動中の詳細画面、またはカード拡大画像）

② iOSショートカットを実行
   → 同じ画像を同じ順番で選択
   → テキストファイルが出力される

③ 神様識別アプリを開く（Safari）
   → 同じ画像を同じ順番で選択
   → アイコン識別結果を確認・修正
   → ショートカットのテキストファイルをアップロード
   → 合体して出力・保存

④ Claudeに貼り付け
   → 手元のヒラメキリストと照合しながらナンバーを特定
   → キャラクター別に振り分け
   → PDFに記録
```

-----

## 注意事項

- ショートカットとアプリで**必ず同じ順番**で画像を選択してください
- 神様識別のスコア差が3.0未満の場合は目視確認を推奨します
- OCRの精度上、テキストに誤認識が含まれる場合があります
- 本ツール群はカオスゼロナイトメア**日本版**専用です
